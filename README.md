# Gut-Friendly Fat-Loss Diet Guide

A personalized 7-day fat-loss and gastritis-safe diet plan web application. The app provides daily meal plans with exact portions, step-by-step preparation instructions, food safety guides, workout meal timing, and a high-protein food reference — all tailored for someone managing gastritis while pursuing fat loss.

## Key Technologies

| Layer | Technology |
|-------|------------|
| Framework | [TanStack Start](https://tanstack.com/start) |
| Frontend | React 19, TanStack Router v1 |
| Build | Vite 7 |
| Styling | Tailwind CSS 4 |
| Language | TypeScript 5.7 (strict mode) |
| Deployment | [Netlify](https://netlify.com) |

## Features

- **7-Day Meal Plan** — Browse each day's breakfast, lunch, snack, and dinner with exact gram/portion measurements
- **Step-by-Step Prep** — Expandable preparation instructions for every meal
- **Safe vs Avoid Foods** — Curated guide of gut-friendly and trigger foods
- **Workout Meal Timing** — Schedule and tips for pre/during/post workout nutrition
- **Protein Guide** — Vegetarian and non-vegetarian high-protein options safe for gastritis

## Running Locally

```bash
npm install
npm run dev
```

The app will be available at [http://localhost:3000](http://localhost:3000).

> Requires Node.js v18+.
